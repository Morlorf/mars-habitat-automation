# User stories

AS operator
I WANT TO read sensors data
SO THAT I can monitor the habitat

AS operator
I WANT TO normalize sensors data
SO THAT I can process them

AS operator
I WANT TO create automation rules
SO THAT I can automate the habitat

AS operator
I WANT TO edit existing automation rules
SO THAT I can modify the behavior of the habitat

AS operator
I WANT TO switch on/off automation rules
SO THAT I can modify the behavior of the habitat

AS operator
I WANT TO save automation rules
SO THAT they persist after a restart

AS operator
I WANT TO interact manually with actuators
SO THAT I can directly control the habitat

AS operator
I WANT TO see the latest sensor readings
SO THAT I can react immediately to critical changes in the habitat.

AS operator
I WANT TO check sensors status
SO THAT I can detect malfunctions
